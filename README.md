# Personal Website

Clean single-page portfolio site for Akshat Singh, focused on hardware, inference systems, and machine learning.

## Stack

- HTML
- CSS
- vanilla JavaScript

## Sections

- Hero with a short positioning statement
- About
- Projects
- Contact

## Local Preview

Open `index.html` directly in a browser, or run a quick static server:

```bash
cd personal-website
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Notes

- The current work line is based on public profile context and can be adjusted easily in `index.html`.
- Styling lives entirely in `styles.css`.
- The JavaScript is intentionally minimal and only powers section reveal motion.
